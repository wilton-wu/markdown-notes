# 1.vs code插件
- Code Runer：运行js代码
- View in Browser：在浏览器打开HTML文件

# 2.vs code快捷键
- Ctrl+Shift+P：显示所有命令
- Ctrl+K Ctrl+O：打开文件夹
- Ctrl+Shift+N：打开新的窗口
- Ctrl+Shift+W: 关闭当前窗口
- Ctrl+K Ctrl+W: 关闭所有标签
- Ctrl+`: 显示隐藏终端
- Ctrl+B: 显示隐藏侧边栏
- Ctrl+]/[: 缩进行
- Ctrl+/: 注释
- Ctrl+D: 查找匹配的下一个
- Ctrl+K V: 预览markdown文件
