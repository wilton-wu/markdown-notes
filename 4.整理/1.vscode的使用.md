# vscode的使用

## 1.vs code插件

- Code Runer：运行js代码
- View in Browser：在浏览器打开HTML文件

## 2.vs code快捷键

- Ctrl+Shift+P：显示所有命令
- Ctrl+K Ctrl+O：打开文件夹
- Ctrl+Shift+N：打开新的窗口
- Ctrl+Shift+W: 关闭当前窗口
- Ctrl+K Ctrl+W: 关闭所有标签
- Ctrl+`: 显示隐藏终端
- Ctrl+B: 显示隐藏侧边栏
- Ctrl+]/[: 缩进行
- Ctrl+/: 注释
- Ctrl+D: 查找匹配的下一个
- Ctrl+K V: 预览markdown文件

## 3.vs code设置vue模板

- Ctrl+Shift+P
- snippets
- vue

```js
{
  "Print to console": {
    "prefix": "vue",
    "body": [
      "<template>",
      "  <div>\n",
      "  </div>",
      "</template>\n",
      "<script>",
      "export default {",
      "  data () {",
      "    return {\n",
      "    }",
      "  }",
      "}",
      "</script>\n",
      "<style lang="less" scoped>\n",
      " ",
      "</style>",
      "$2"
    ],
    "description": "Log output to console"
  }
}
```

## 4.vs code设置成中文

- Ctrl+Shift+P
- language
- 安装中文简体语言包

## 5.vs code合并分支

- Ctrl+Shift+P
- git merge
- 选择要合并的分支
- 推送到当前分支
- 图形化工具：SourceTree

## 6.vs code配置bash终端

- Ctrl+Shift+P
- setting

```js
"terminal.integrated.shell.windows": "D:\\git\\bin\\bash.exe"
```
